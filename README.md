# Login-Register Project

A simple Flask-based Login and Registration system built using Flask, HTML, and CSS.

## Features

- User Registration
- User Login
- Home Page
- File Upload Feature

## Installation

### Clone the repository

```bash
git clone https://github.com/aman-shh/login-register-project.git
```

### Go to the project folder

```bash
cd login-register-project
```

### Install dependencies

```bash
pip install flask
```

### Run the application

```bash
python app.py
```

### Open in browser

```text
http://127.0.0.1:5000
```

## Technologies Used

- Python
- Flask
- HTML
- CSS
